# Notefy !
Simple Python based notes app for organizing your notes. Made with Tkinter python3. 

![Static Badge](https://img.shields.io/badge/GitHub-Notefy-blue?logo=github) ![Static Badge](https://img.shields.io/badge/Python-3.13.7-yellow?logo=python)

## I) Feautures:
- Simple to use.
- Fast.
- Reliable.
- ***100%*** Customisable.
- All in one Json file.
## II) Installation:
### 1. Install the latest verion of **Python**:
### Installation link:
Download the latest version of python for window from here: 
https://www.python.org/downloads/
### Or, For Linux/MacOs:
You can use the following commands for the spefic distro/os that you use:
#### Linux:
- *Ubuntu/Debian:*
```bash
sudo apt update && sudo apt install python3
```
- *Arch:*
```bash
sudo pacman -S python
```
- *Fedora:*
```bash
sudo dnf install python3
```
- *CentOS/RHEL:*
```bash
sudo yum install python3
```
- *openSUSE:*
```bash
sudo zypper install python3
```
- *Slackware:*
```bash
sudo slackpkg install python
```
### MacOs:
- *Homebrew:*
```bash
brew install python
```
- *MacPorts:*
```bash
sudo port install python3
```
### 2. Install Notefy:
- By Installing the ZIP file: https://github.com/Shaedow2000/Notefy/archive/refs/heads/main.zip
- Or cloning the repo:
```bash
git clone https://github.com/Shaedow2000/Notefy.git
```
### 3. Start the program:
Just execute the main python file:
```bash
python main.py
```
## III) Contribution:
All contributions are welcome !
#### To contribute to this project:
- Fork the project.
- Clone the repo localy:
```bash
git clone https://github.com/Shaedow2000/Notefy.git
```
- Set upstream to this repo:
```bash
git remote add upstream https://github.com/Shaedow2000/Notefy.git
```
- And to check if it works:
```bash
git remote -v
```
You will see this:
```
> origin    https://github.com/YOUR-USERNAME/YOUR-FORK.git (fetch)
> origin    https://github.com/YOUR-USERNAME/YOUR-FORK.git (push)
> upstream  https://github.com/Shaedow2000/Notefy.git (fetch)
> upstream  https://github.com/Shaedow2000/Notefy.git (push)
```
- Now start coding your changes !!
